🌿 Jeevandhaara – Rural Telemedicine Platform

Jeevandhaara is a web-based telemedicine platform developed by Team Tech4Good to bridge the healthcare accessibility gap in rural communities.

Many patients in remote areas must travel long distances even for basic medical consultations. Jeevandhaara brings healthcare closer by enabling remote doctor consultations, secure medical record sharing, and instant digital prescriptions through a simple and accessible web interface.

The platform follows a human-centered, low-bandwidth-first design, ensuring usability even in areas with limited connectivity and basic devices.

By digitizing the consultation process and creating a structured virtual clinic experience, Jeevandhaara makes healthcare more accessible, efficient, and reliable for underserved populations.

🚀 Key Features
🩺 Unified Patient–Doctor Portal

Patients can schedule and attend remote consultations with doctors through a secure web interface.

📄 Secure Medical Report Management

Patients can upload diagnostic reports and medical records, allowing doctors to access them instantly during consultations.

💊 Automated Digital Prescription System

Doctors can generate secure digital prescriptions that patients can download immediately after consultation.

🏥 Virtual Clinic Workflow

Integrates appointment scheduling, consultation, report sharing, and prescriptions into one seamless digital healthcare process.

📶 Low-Bandwidth Friendly Design

Optimized to function smoothly even on slow internet connections and basic devices, making it ideal for rural environments.

⚙️ Installation & Setup

Clone the repository

git clone https://github.com/ShreyaaaSh12/JeevanDhaara_telemedicine.git

Navigate to the project directory

cd JeevanDhaara_telemedicine

Install dependencies

npm install

Start the development server

npm start

Open in browser:

http://localhost:3000
